# Fonts-xkcd
Fonts required for xkcd in Matlplotlib

1. Download fonts to

    ````
    /usr/share/matplotlib/mpl-data/fonts/ttf
    ````
2. Delete font cache

    ````
    rm ~/.cache/matplotlib/fontList.cache
    ````
3. Restart jupyter
